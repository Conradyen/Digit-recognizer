Digit Recognizer

A python implementation of neural net deploy on AWS. The model is trained on kaggle digit recognizer data set a trim version of MNIST (42000 training images). architecture of the model follows 5 layer LeNet with slightly change. The model achieves 99.2% accuracy on kaggle leader board after 50 epoch. More details are on github.

Since the model is train on images with size 28 by 28 the input digit has to be preprocess before feed into the neural net. The hand written digits first being invert and centered by cropping out all columns and rows that are all zeros. Padded to 280 by 280 and down sampled to 28 by 28 pixels before the model can make prediction.
